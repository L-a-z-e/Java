형변환 (Casting) - 변수 또는 상수의 타입을 다른 타입으로 변환하는 것
(Type)피연산자

정수형간의 형변환
int -> byte Casting 시 loss of data

byte -> int  남은 빈공간은 0 (양수) 1 (음수) 로 채움

실수형간의 형변환

작은타입에서 -> 큰 타입으로 변환하는 경우 빈공간 0으로 채움

double 타입에서 float 타입으로 변환하는 경우 double의 가수 52자리중 23자리는 저장되고 나머지는 버려짐
* 24번째 자리에서 반올림이 발생할 수 있음
* float 타입의 범위를 넘는 값을 float로 변환하는 경우 +- 무한대 or +- 0

정수형 -> 실수형
float => 7자리 정밀도
int의 최대값은 20억 10자리의 정밀도 필요
그래서 8자리 이상의 정수형을 실수형으로 변환할때는 double 사용

실수형 -> 정수형으로 변환
소수점 이하 값은 버려짐 (반올림이 아니라 버림)

자동 형변환
컴파일러가 생략된 형변환을 자동적으로 추가한다
ex ) float f =1234; -> float f = (float)1234;

byte b =1000; // 범위 -128~127 초과
incompatible type 에러발생

char ch = (char)1000; // 명시적 형변환으로 에러발생안함

int i = 3;
double d = 1.0 + i ; // double d = 1.0 + (double)i 로 자동형변환

자동 형변환 규칙 -> 기존 값을 최대한 보존할 수 있는 타입으로 자동 형변환
기본형과 참조형은 서로 형변환할 수 없다.
값의 범위가 작은 타입에서 큰 타입으로의 형변환은 생략할 수 있다.
